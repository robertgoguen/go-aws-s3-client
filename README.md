# aws-go-s3-client

## A Simple AWS S3 Client, written in Golang

### Requires aws-cli
* apt-get -y update
* apt-get -y upgrade
*
* apt-get install -y python3-pip
*
* python3 -m venv env
*
* source ./env/bin/activate
*
* pip3 install awscli
*
* pip3 install localstack
*
* SERVICES=sqs,sns,s3,lambda DEBUG=1 localstack start
